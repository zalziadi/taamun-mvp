جدول مهام شاهين المالية

كل مهمة مرتبطة بالمنصة والتكرار ونوع الأتمتة
المهمة المنصة التكرار النوع المخرج
مزامنة المنتجات مع سلة تمعّن + مسخّر → سلة عند التغيير أتمتة كاملة منتج محدّث في سلة
معالجة الدفع الناجح سلة → شاهين فوري (Webhook) أتمتة كاملة تفعيل الخدمة + إشعار
إصدار كود التفعيل (تمعّن) شاهين → تمعّن بعد الدفع أتمتة كاملة كود TAAMUN-XXX للعميل
فتح وصول الكورس (مسخّر) شاهين → مسخّر بعد الدفع أتمتة كاملة وصول مفعّل + رسالة ترحيب
تجديد الاشتراكات سلة → شاهين دوري (حسب الباقة) أتمتة كاملة تمديد الوصول أو تعليقه
معالجة الاسترداد سلة → شاهين → البنك عند الطلب أتمتة + موافقة استرداد مبلغ + إلغاء وصول
التسوية اليومية سلة + البنك يومي أتمتة كاملة تقرير مطابقة
تقرير الإيرادات كل المنصات أسبوعي / شهري أتمتة تقرير مالي شامل
تنبيه الفروقات شاهين → زياد عند الكشف أتمتة + إشعار تنبيه فوري بالفرق

تنسيق شاهين مع الإيجنتات الأخرى

كيف يتواصل شاهين مع باقي إيجنتات المنظومة

🦅 شاهين → إيجنت المبيعات

  • إرسال تأكيد الدفع الناجح
  • تحديث حالة العميل (مدفوع/معلّق)
  • إشعار بالاشتراكات القريبة من الانتهاء

🦅 شاهين → إيجنت المحتوى

  • فتح الوصول للمحتوى بعد الدفع
  • إغلاق الوصول عند انتهاء الاشتراك
  • تفعيل المحتوى الإضافي (باقة 820)

🦅 شاهين → إيجنت التواصل

  • إرسال إيصال الدفع للعميل
  • تذكير بموعد التجديد
  • إشعار بنجاح الاسترداد

🦅 شاهين → زياد (التقارير)

  • ملخص مالي أسبوعي
  • تنبيه فوري عند أي مشكلة مالية
  • تقرير شهري شامل بالإيرادات

مهمة شاهين: تفعيل بوابات الدفع عبر سلة

خطوات الربط الكامل بين تمعّن ومتجر سلة

١

إنشاء المنتجات على سلة

أنشئ ٤ منتجات في متجر سلة تُمثّل الباقات: عيدية (28 ر.س) — شهري (82 ر.س) — سنوي (820 ر.س) — VIP (8,200 ر.س)

٢

إضافة متغيرات البيئة على Vercel

PAYMENT_PROVIDER=salla
SALLA_PRODUCT_EID=https://store.salla.sa/.../product/...
SALLA_PRODUCT_MONTHLY=https://store.salla.sa/.../product/...
SALLA_PRODUCT_YEARLY=https://store.salla.sa/.../product/...
SALLA_PRODUCT_VIP=https://store.salla.sa/.../product/...
SALLA_WEBHOOK_SECRET=... (للـ webhook)

٣

تفعيل Webhook على سلة

من لوحة تحكم سلة → الإعدادات → Webhooks → أضف:
https://taamun-mvp.vercel.app/api/salla/webhook
الأحداث: order.created, order.payment.updated, order.updated

٤

Deploy + اختبار

انشر التعديلات على Vercel → اختبر عملية الشراء من الباقة الشهرية → تأكد أن الـ Webhook يُفعّل الاشتراك

٥

تنسيق مع مسخّر

نفس المتجر على سلة يمكن أن يضم منتجات مسخّر — نفس الـ Webhook يعالج كل المنصات
