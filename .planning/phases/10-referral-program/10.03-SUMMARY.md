## Response Contract

| Status | Body                                                              | When                                                                                      |
| ------ | ----------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| 200    | `{ ok: true, code: "FRIEND-XXXXXX", reused: false }`              | New code minted and inserted                                                              |
| 200    | `{ ok: true, code: "FRIEND-XXXXXX", reused: true }`               | User already had a `pending_invitee` row — same code returned, no insert, no event        |

| 500    | `{ ok: false, error: "server_misconfig" }`                        | `getSupabaseAdmin()` threw (missing `SUPABASE_SERVICE_ROLE_KEY`)                          |
| 500    | `{ ok: false, error: "server_error" }`                            | Cap-check query failed, code-gen exhausted, or INSERT failed                              |

## Cap-Check SQL (literal from `route.ts:81-91`)

```ts
const startOfYear = new Date(
  Date.UTC(new Date().getUTCFullYear(), 0, 1),
).toISOString();

const { count, error: countError } = await admin
  .from("referrals")
  .select("id", { count: "exact", head: true })
  .eq("referrer_id", auth.user.id)
  .eq("status", "rewarded")
  .gte("created_at", startOfYear);
```
