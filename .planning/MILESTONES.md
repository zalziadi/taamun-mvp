## Milestone philosophy

Each milestone = one shippable slice of value. Between milestones, the codebase is always deployable.

- **v1.0:** Make it work (core loop) ✅
- **v1.1:** Make it sticky (retention + community) ✅
- **v1.2:** Make it personal (depth + AI customization) ✅
- **v1.3:** Make it spreadable (reach + distribution)
- **v2.0:** Make it a platform (multi-user, creator-driven)
