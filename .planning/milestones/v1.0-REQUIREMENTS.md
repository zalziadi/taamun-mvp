# Requirements — v1.0 (Archived)

All requirements from v1.0. Status marked based on actual shipped features.

---

## User experience

- [x] New users see philosophical "/welcome" page before signup
- [x] 28-day program with daily verse, silence moment, exercise, reflection
- [x] Streak tracking with visual indicator
- [x] Progress percentage + completion milestones
- [x] Reflection journal auto-saves
- [x] Awareness meter evolves after day 10 (deeper question)
- [x] Smart Paywall: first 3 days fully open, single paywall after

## AI guide

- [x] Guide personality: name "تمعّن", warm friend tone
- [x] Dynamic greeting based on journey stage (5+ variations)
- [x] 6-stage conversation framework enforced via system prompt
- [x] Daily message limit (5/day) for trial users
- [x] Unlimited for paid subscribers
- [x] RAG mode connected to book content
- [x] Voice session (`/guide/voice`) via Munsit
- [x] Post-28 "continuous companion" mode

## VIP experience

- [x] Gene Keys profile (11 spheres) calculated from birth data
- [x] BaZi integration (day master, dominant element)
- [x] No-assumption protocol in VIP prompt
- [x] Last-topic greeting referencing user_journey.last_insight
- [x] Dynamic quick prompts from Gene Keys spheres
- [x] Dedicated `/profile/map` page
- [x] VIP navbar badge visible on all pages

## Validation

**First real customer testified on day 9:**
> "مشيت فيه لليوم التاسع. حبيت البرنامج وخليته من روتين الصبح. أعطي ٥ دقائق تأمل — جداً تفرق. أحس قلبي يتشرب معاني."

This validates:
- ✓ 5-minute daily format works
- ✓ Morning routine integration succeeds
- ✓ Emotional depth reached ("قلبي يتشرب معاني")
- ✓ Retention through day 9 (past critical dropoff at day 3)

v1.0 design thesis: **proven**.
