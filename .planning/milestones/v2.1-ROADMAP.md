- Amber hero banner when any flagged items exist

- Admin dashboard now surfaces an "نظرة تشغيلية" card alongside the moderation card

## Non-functional compliance

- Zero new dependencies, zero new migrations
- All reads are `head: true` → cheap, no row materialization
- Admin-only gate
- Manual refresh only (no auto-poll) — avoids running up usage during pre-validation
