## Shipped phases

### Phase 1: Journey OG image generator
- `src/app/journey/[slug]/opengraph-image.tsx` — next/og ImageResponse at 1200×630
- Fonts: Amiri Regular + Noto Naskh Arabic Bold, fetched at request time
- Layout: brand header / large RTL title / author + subscriber count footer
- Mirrors the v1.4 `/shared/[slug]` generator for visual consistency

### Phase 2: JSON-LD structured data
- `src/lib/json-ld.ts` — builders for Organization, Course, DiscussionForumPosting, FAQPage
- Global Organization schema in `layout.tsx` head
- Course schema in `/journey/[slug]` page (published only)
- DiscussionForumPosting schema in `/threads/[id]`
- FAQPage schema in `/faq` (ships below)

### Phase 3: Creator onboarding guide
- `src/app/creator/guide/page.tsx` — 5-step public guide (VIP → draft → days → publish → analytics)
- Each step has a body + 2–4 concrete bullets
- CTA buttons linking to `/creator` and `/discover`
- Linked from sitemap + robots

### Phase 4: FAQ page + FAQPage schema
- `src/app/faq/page.tsx` — 8 Q&A pairs covering program, pricing, trial, creator mode, invites, privacy, refunds, offline
- Embedded FAQPage JSON-LD via `faqSchema()` from lib
- CTA linking to `/guide` when questions remain
- Linked from sitemap + robots

## Non-functional compliance

- Zero new dependencies, zero new migrations
- OG generator degrades gracefully when fonts fetch fails
- JSON-LD injection uses `jsonLdString()` which escapes `</` to prevent HTML escape
- All new pages RTL + indexable
- Course schema only emitted for published journeys (draft stays private)
