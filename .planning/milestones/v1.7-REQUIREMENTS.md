# Requirements — v1.7 (archived)

Snapshot of v1.7 requirements as shipped 2026-04-20.

---

## Mission

Deepen the creator graph: public creator profiles, following surface for
members, smarter discovery ranking, and a visible admin queue badge.

---

## Must-have ✅

- ✅ Public creator profile at `/creator/by/[userId]`
- ✅ "Following" tab in `/account`
- ✅ `/discover` sorted by subscriber_count DESC (+ top row)
- ✅ Admin mod queue hero badge

---

## Open questions — resolved

1. **Profile gating:** ✅ public (anon-readable)
2. **Following tab empty state:** ✅ hidden when no follows (not shown as empty list)
3. **Discover top count:** ✅ top 3 with traction (subscriber_count > 0)
4. **Badge location:** ✅ /admin hero area, not mobile nav

---

## Non-functional

- Zero new dependencies, zero new migrations
- All new surfaces RTL Arabic
- Public profile indexable (generateMetadata + OG)
