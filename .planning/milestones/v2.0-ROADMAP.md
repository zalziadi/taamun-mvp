# Milestone v2.0 — Internationalization

**Status:** ✅ Shipped
**Shipped:** 2026-04-20
**Commits:** 1 (v2.0 — all 4 phases batched)

---

## Mission

Open Taamun to expat Muslims in English-speaking countries **without
touching the core Arabic experience**. Landing + FAQ + switcher + proper
hreflang signaling. Arabic remains canonical; English is a marketing
bridge pointing back to the Arabic app.

---

## Shipped phases

### Phase 1: English landing at `/en`
- `src/app/en/layout.tsx` — LTR nested layout with `lang="en"`
- `src/app/en/page.tsx` — static marketing page (pitch, 3 feature cards, testimonial quoting the real Arabic feedback, "who is this for", 7-step program breakdown, CTA to Arabic app)
- Metadata + hreflang alternates linking back to `/`
- Clear disclosure: "The experience itself is in Arabic."

### Phase 2: English FAQ at `/en/faq`
- `src/app/en/faq/page.tsx` — 8 Q&A pairs translated from `/faq`
- `FAQPage` JSON-LD via existing `faqSchema()` builder
- Metadata + hreflang alternates linking back to `/faq`
- CTAs to Arabic app + Arabic FAQ

### Phase 3: hreflang + sitemap + robots
- Root `layout.tsx` metadata: `alternates.languages` adds `ar`, `en`, `x-default`
- `/faq` page: explicit `alternates.languages` mirrors
- `sitemap.ts`: `/en` (0.8) and `/en/faq` (0.6) added
- `robots.ts`: `/en` + `/en/` allowed

### Phase 4: Language switcher
- `src/components/LanguageSwitcher.tsx` — context-aware AR ↔ EN toggle
  - On `/` → links to `/en` (label "English")
  - On `/en` → links to `/` (label "العربية")
  - On `/faq` → links to `/en/faq`, and vice versa
  - Any other path without a mirror falls back to language home
- Mounted in `AppChrome` header nav, next to `NavAuth`

---

## Non-functional compliance

- Zero new dependencies, zero new migrations
- `/en` and `/en/faq` are fully static (pre-rendered on CDN)
- Arabic experience untouched — no direction flip, no layout regression
- Switcher emits proper `hreflang` attribute on the link
- Metadata canonical / alternate chain complete for Google i18n signals
