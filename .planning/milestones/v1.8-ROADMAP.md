# Milestone v1.8 — Growth & SEO Surfaces

**Status:** ✅ Shipped
**Shipped:** 2026-04-20
**Commits:** 1 (v1.8 — all 4 phases batched)

---

## Mission

v1.4–v1.7 built the creator graph. v1.8 pushes it outward:
real-user social proof on marketing surfaces, a featured creator
journey for logged-out visitors, sitemap + robots for SEO, and a
public leaderboard for top creators.

---

## Shipped phases

### Phase 1: Social proof rotator
- `src/components/SocialProofRotator.tsx` — client component polling `/api/shared`, rotates 3 real published insights every 6s with dot indicators
- `src/app/api/shared/route.ts` — new GET endpoint (limit=N, anon-readable via RLS)
- Wired into homepage (between landing + authed island) + `/pricing` (below founder testimonial)
- Silent no-op when no published insights exist

### Phase 2: Featured creator journey spotlight
- `src/components/FeaturedJourney.tsx` — client spotlight card linking to the top journey
- `src/app/api/discover/featured/route.ts` — returns single most-subscribed published journey
- Wired into homepage above the social proof rotator
- Silent when no journey has been published yet

### Phase 3: sitemap.xml + robots.txt
- `src/app/sitemap.ts` — `MetadataRoute.Sitemap` generator, force-dynamic, adds static paths + published journeys, unique creator profiles, published threads, shared insights; try/catch falls back to static entries on DB failure
- `src/app/robots.ts` — allows public pages + `/shared/`, disallows `/admin`, `/account`, `/api`, `/auth`, `/checkout`; declares sitemap URL + host

### Phase 4: Public creator leaderboard
- `src/app/creator/leaderboard/page.tsx` — SSR top 10 creators by aggregate subscribers across all their published journeys; shows journey count + title of top entry
- Linked from sitemap + accessible as a public SEO surface
- Empty state preserves the UX when the community is new

---

## Non-functional compliance

- Zero new dependencies, zero new migrations
- All new client components silent on empty data (marketing pages don't show empty ugly cards)
- Sitemap + featured both degrade to sensible defaults on DB error
- `/creator/leaderboard` indexable; SEO metadata + OG
- Robots explicitly blocks admin/account/api/auth/checkout paths
