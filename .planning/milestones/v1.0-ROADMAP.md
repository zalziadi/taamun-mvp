## Mission

Build a shippable Quranic contemplation app that transforms reading into experience through a 28-day guided journey.

---

## Shipped phases

### Phase 0: Foundation (Feb 14 — Mar 31)

Infrastructure, Next.js setup, Supabase integration, auth flow, basic UI shell.

**Deliverables:**
- Next.js 14 App Router + Supabase auth (magic link + phone OTP)
- Brand identity (fonts, colors, Arabic RTL throughout)
- 54 pages scaffolded
- 77 API endpoints
- 30 DB tables via Supabase migrations

### Phase 1: Core Program (Mar — early Apr)

The 28-day journey experience.

**Deliverables:**
- 28 days of verse content + reflection questions + exercises
- DayExperience component (7-layer daily ritual)
- Silence moment → Verse → Hidden layer → Book quote → Exercise → Reflection journal → Awareness meter
- Progress tracking with streak calculation
- Auto-save reflection journal (Supabase)

### Phase 2: AI Guide "تمعّن" (Apr)

Companion that knows the user's journey.

**Deliverables:**
- 6-stage conversation framework (احتواء → استكشاف → لحظة → استخراج → هدية → خطوة)
- Anthropic Claude integration
- Dynamic greeting based on day + session type
- RAG mode connected to book chunks
- Voice session via Munsit (STT)
- Post-28 "continuous companion" mode

### Phase 3: VIP Experience (Apr)

Exclusive depth for highest-tier subscribers.

**Deliverables:**
- Gene Keys profile (11 spheres + Quranic verse mappings)
- BaZi birth chart integration
- VIP system prompt with "no-assumption protocol"
- Last-topic greeting ("آخر مرة تكلمنا عن...")
- Dynamic quick prompts from user's Gene Keys spheres
- `/profile/map` dedicated page
- VIP navbar badge

### Phase 4: Post-28 Retention (Apr 15-16)

Content + UX for users who complete the 28 days.

**Deliverables:**
- Completion celebration ("أتممت الرحلة")
- Cycle 2 (النفس والعلاقات) — 28 new days
- Cycle 3 (السور الكاملة) — 28 new days
- Cycle-aware content selector + DB schema (current_cycle, completed_cycles)
- Weekly challenge system (4 challenges, cycling)
- Daily verse rotation (30 verses, date-based)
- "Days since completion" counter with contextual messages
- Journey milestones (28, 60, 90, 180, 365)

### Phase 5: Performance & UX (Apr 16-17)

Static homepage, server components, performance optimization.

**Deliverables:**
- Static homepage (CDN-cached, zero cold start)
- JourneyLanding → Server Component refactor
- next/font/google for all fonts (CLS fix)
- Lighthouse A11y 100, SEO 100
- Code cleanup: removed 2,161 lines of dead code
- TTFB 600ms → 110ms (-82%)
- CLS 0.063 → 0.005 (-92%)

### Phase 6: Community & Book (Apr 17)

Community integration + book reader enhancements.

**Deliverables:**
- CommunityPulse (live active users + shared reflections)
- CommunityJoin (WhatsApp CTA, dismissable)
- BookProgress (14-chapter tracker, localStorage)
- BookTools (bookmarks + suggested chapter)
- Pricing page: customer testimonial from first real user

### Phase 7: Retention & Minimal UI (Apr 18)

Continuity features + user-requested visual flattening.

**Deliverables:**
- ReflectionEcho (shows user's past reflection, "قلبك كتب قبل أسبوع")
- ShareableVerse (screenshot-friendly card for social sharing)
- Streak protection (3 AM grace period, not midnight)
- Time-aware greeting (morning/evening/late-night)
- Retention messages on home ("أغلب من بدأ توقّف بيوم ٣")
- **Minimal flat UI redesign** (user-requested):
  - Cards: removed boxes/shadows/borders → 3 subtle horizontal lines
  - Buttons: removed shadows + translate animations
  - Header/footer/mobile nav: removed backdrop-blur + opacity
  - Paywall: rounded boxes → border-y lines
  - Welcome + Account pages: cleaned
