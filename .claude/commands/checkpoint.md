احفظ نقطة تراجع الآن قبل البدء بتعديل خطر.

1. شغّل `git status` لتعرف ما الذي سيُحفظ
2. شغّل `git add -A`
3. شغّل `git commit -m "chore: checkpoint — $ARGUMENTS"` (إذا لم يُعطَ اسم استخدم "before changes")
4. أخبر المستخدم: "✓ checkpoint محفوظ. يمكنك الرجوع إليه بـ `git reset --hard HEAD~1` إذا احتجت."
