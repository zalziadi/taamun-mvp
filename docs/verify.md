# docs/verify.md — خطوات التحقق

## التحقق التلقائي (npm run verify)

1. **typecheck**: `tsc --noEmit` — لا أخطاء TypeScript
2. **build**: `next build` — البناء ينجح

## التحقق اليدوي (للاختبار البشري)

| # | الخطوة | النتيجة المتوقعة |
|---|--------|------------------|
| 1 | `npm run dev` | الخادم يشتغل على localhost:3000 |
| 2 | فتح `/` | الصفحة الرئيسية تعرض روابط |
| 3 | فتح `/day` | صفحة اليوم تعرض آية + أسئلة + chips |
| 4 | اختيار مرحلة + حفظ | يظهر "تم الحفظ ✅" |
| 5 | فتح `/progress` | شبكة 28 يوم + إحصاءات |
| 6 | فتح `/subscribe` | السعر 280 + زر واتساب |
| 7 | Refresh الصفحة | البيانات لا تضيع (LocalStorage) |

## شروط النجاح
- لا crash في أي صفحة
- RTL مفعّل (الاتجاه من اليمين)
- التخزين يعمل بعد الـ refresh
