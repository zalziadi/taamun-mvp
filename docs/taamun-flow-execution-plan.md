# تمعّن — وثيقة الفلو الكامل وخطة التنفيذ

**الهدف:** 28 مشترك من 55,000 زائر خلال شهرين (معدل تحويل 0.05%)
**استراتيجية البيع:** خيارين فقط — العيدية (28 ر.س) + سنوي (820 ر.س)
**بوابات الدفع:** Tap Payments + تحويل بنكي يدوي
**مصادر الزوار:** مزيج (إنستغرام + تيك توك + واتساب + تلغرام + إعلانات)

> **ملاحظة استراتيجية:** معدل التحويل المطلوب (0.05%) منخفض جداً.
> التحدي الحقيقي ليس التحويل بل **جلب 55 ألف زائر في شهرين**.
> مع فلو محكم، المعدل الواقعي 1-3% = 550-1,650 مشترك محتمل.
> **الفرصة الحقيقية: إذا وصل 55 ألف، النتيجة ستكون أضعاف الـ 28.**

## ٢. العيدية — آلية العمل

### ما هي؟
عرض دخول منخفض التكلفة (28 ر.س) يعطي المستخدم تجربة محدودة (3 أيام) ليعيش الفلو ثم يقرر الترقية.

### كيف تعمل تقنياً؟
- صفحة `/eid` موجودة وتعمل — تجربة 3 أيام مع عرض ترقية
- الدفع عبر Tap أو تحويل بنكي
- كود التفعيل: `TAAMUN-280-XXX` (باقة 280 المخفّضة)
- بعد انتهاء الـ 3 أيام: يظهر عرض الترقية للسنوي

### الفلو المقترح للعيدية:

```
المحتوى (ريل/بوست) → "جرّب تمعّن — 28 ريال فقط"
    │
    ▼
صفحة العيدية (/eid) — 3 أيام تجريبية
    │  ├─ يوم ١: آية + صمت + تأمل
    │  ├─ يوم ٢: آية + طبقة مخفية + يوميات
    │  ├─ يوم ٣: آية + مقياس الوعي + بطاقة مشاركة
    │  └─ نهاية اليوم ٣: "هل تريد إكمال الـ 28 يوم؟"
    │
    ▼
عرض الترقية ← 820 ر.س/سنة
    ├─ كل ما في العيدية +
    ├─ 25 يوم إضافي
    ├─ الكتاب الكامل
    ├─ المرشد الذكي
    ├─ السبحة
    └─ مدينة المعنى التفاعلية
```

### الحالة: ✅ صفحة `/eid` تعمل — تحتاج ربط الدفع بـ Tap فقط

- اختيار الأنسب أو الدمج بينها
- ربطها كصفحة رئيسية (/)

### الخطوة ٣: ضبط صفحة التسعير (يوم واحد)
- تبسيط إلى خيارين فقط: العيدية (28) + سنوي (820)
- إزالة الباقات الأخرى مؤقتاً لمنع التشتت
- ضبط CTAs: "جرّب 3 أيام" + "ابدأ عامك الكامل"

### الخطوة ٤: التسخين (8 أسابيع)
| الأسبوع | النشاط | الهدف |
|---|---|---|
| ١-٢ | محتوى عميق (ريلز + بوستات) عن التمعّن | بناء الوعي |
| ٣-٤ | قصص تجريبية + "ماذا يحدث في 3 أيام" | بناء الفضول |
| ٥-٦ | فتح العيدية كـ "دعوة محدودة" | أول 15 مشترك |
| ٧-٨ | شهادات + عرض الترقية السنوية | تحويل + 13 مشترك |

### الخطوة ٥: المتابعة والتحويل
- متابعة المشتركين عبر واتساب
- إرسال تذكير يومي بسيط
- عرض الترقية بعد اليوم ٣ للعيديات
