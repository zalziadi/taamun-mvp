# مسح الآية (Ayah Scan) — قائمة التحقق اليدوي

## 1. التحقق من الوصول

- [ ] بدون خطة 820: `/scan` يعرض "خطة 820 مطلوبة" + رابط الترقية
- [ ] مع خطة 820: صفحة المسح تظهر (اختر صورة، استخراج النص، ابدأ التمعّن)
- [ ] للتجربة: `localStorage.setItem("taamun.plan820.v1","1")` + الاشتراك النشط (أو أدمن)

## 2. OCR

- [ ] اختيار صورة تحتوي نص عربي
- [ ] الضغط على "استخراج النص"
- [ ] يظهر النص في textarea (قابل للتعديل)
- [ ] في حال فشل OCR: تظهر رسالة خطأ واضحة

## 3. جلسة التمعّن

- [ ] بعد تعديل النص، الضغط على "ابدأ التمعّن"
- [ ] الانتقال إلى `/day?mode=scan`
- [ ] ظهور "الآية المستخرجة" + النص + أسئلة (ظل، إدراك، تمعّن)
- [ ] اختيار مرحلة + حفظ
- [ ] ظهور "تم حفظ جلسة اليوم ✅"
- [ ] الجلسة لا تقفل اليوم العادي (اليوم الرئيسي يبقى مفتوحاً)

## 4. الكلمات المفتاحية

- [ ] آية تحتوي "الله أحد" أو "الصمد" → أسئلة التوحيد
- [ ] آية تحتوي "الشيطان" أو "المال" → أسئلة التعلق
- [ ] آية تحتوي "رحيم" أو "توبة" → أسئلة الرحمة
- [ ] آية تحتوي "رزق" أو "بركة" → أسئلة الرزق
- [ ] آية بدون تطابق → الأسئلة الافتراضية

## 5. التخزين

- [ ] `tmn.scan.ayahText.v1` في localStorage يحتوي النص بعد "ابدأ التمعّن"
- [ ] النص لا يُحذف تلقائياً بعد الجلسة
