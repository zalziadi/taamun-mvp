# Landing V2 Content Blueprint

## Purpose
- Reframe `landing-2` as a conversion-focused page centered on the Taamun book journey.
- Position Taamun as a full product ecosystem, not a single offer.
- Clarify price anchors so users understand the path from free entry to paid depth.

## Hero Message
- Core idea: this is not a motivational page, it is a structured transformation path.
- Promise: practical inner-work framework with clear next steps.
- CTA direction: one primary action, one low-friction secondary action.

## Book-Led Narrative
1. Start with the emotional pain: confusion, drift, and decision fatigue.
2. Introduce the book as the first stable reference point.
3. Explain what the user gets immediately after reading.
4. Bridge to the broader Taamun ecosystem for continuity and accountability.

## Product Ecosystem Pricing
- Entry Layer: book + foundational guidance.
- Practice Layer: reflective tools and routine-based implementation.
- Growth Layer: premium programs for deeper transformation.

### Pricing Intent
- Use clear anchors to reduce perceived risk.
- Show value stacking before showing total package price.
- Keep each tier outcome-based, not feature-based.

## Section Order (Landing 2)
1. Hero (problem + promise + CTA)
2. Credibility and lived proof
3. Book outcomes and core themes
4. Ecosystem map (what comes after the book)
5. Pricing and offer architecture
6. FAQ and final conversion block

## Copy Rules
- Use direct Arabic with high emotional clarity and low abstraction.
- Keep headlines short and thesis-driven.
- Every section must answer: "What changes for me after this?"
- Avoid generic hype language.

## CTA Strategy
- Primary CTA: start now / secure access.
- Secondary CTA: read more details for users who need certainty.
- Repeat CTA after every major value block.

## Notes For Iteration
- Track scroll depth and CTA click-through by section.
- If drop-off happens before pricing, tighten narrative transition from book to ecosystem.
- If clicks are high but purchases are low, simplify the offer comparison language.
