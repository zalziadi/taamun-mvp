## 1. Landing Page — الصفحة الرئيسية (Desktop)

```
Design a premium Arabic RTL landing page for "تمعّن" (Taamun) — a 28-day Quranic contemplation app.

THEME: Ultra-dark background (#15130f), gold/parchment accents (#f0e1c0, #c4a882), minimal and spiritual, lots of negative space. Serif Arabic font (Amiri) for headings, thin sans-serif (Manrope 300) for body. No bright colors — everything muted and sacred.

LAYOUT:
- Fixed header: logo "تمعّن" on the right, minimal nav links on left
- Hero: centered vertical staff illustration (thin gold line with diamond top), ambient glow behind it, 9 small stones in arc below
- Quranic verse in large Amiri font: "وَتَوَكَّلْ عَلَى الْحَيِّ الَّذِي لَا يَمُوتُ"
- Small English translation below in uppercase tracking
- CTA button: "ابدأ رحلتك اليوم" with gold border
- Journey progress bar: thin line from "ظِل" to "أفضل احتمال"
- 9 domains grid (3×3): الهوية، العلاقات، المال، النمو، العطاء، الجمال، الأسرة، البناء، المراجعة — each with icon, name, description
- Metaphor section: split layout, sticky quote on right, 4 items on left (العصا، الظل، النور، الأحجار التسعة)
- Social card: 1:1 square with verse, dark background, radial glow
- Footer: minimal with "تمعّن • العتبة الصامتة • ١٤٤٧"

INTERACTIONS: Custom cursor (dot + ring), scroll-linked color transition from gold to white, scroll reveal animations. Grain texture overlay.
```

## 2. Day Experience — تجربة اليوم (Desktop)

```
Design a contemplative daily practice screen for "تمعّن" — a Quranic meditation app. Arabic RTL.

THEME: Same dark palette (#15130f background, #f0e1c0 gold text). Extremely minimal. The screen should feel like entering a sacred space, not reading a webpage.

LAYOUT — vertical scroll, one element at a time:
1. SILENCE GATE: Full-screen dark with only "اصمت" in small gold text center, fades after 5 seconds
2. VERSE BLOCK: Large Quranic verse in Amiri serif, centered, with ambient glow. Surah reference below
3. HIDDEN LAYER: Brief contemplation prompt, initially hidden, revealed on tap/click
4. BOOK QUOTE: Elegant blockquote from "مدينة المعنى" book, with thin gold border-right
5. REFLECTION JOURNAL: Clean textarea with placeholder "ماذا رأيت؟", auto-saves
6. AWARENESS METER: Horizontal scale 1-10, gold dots, label "مستوى الحضور"
7. SHARE CARD: Generated 1:1 card with today's verse for sharing
8. PROGRESSION BADGE: Circular badge showing day number (e.g. "اليوم ٧") with milestone indicator

Typography: Amiri for verses/quotes, Manrope thin for UI elements. No borders, no boxes — just space and text.
```

## 3. Pricing Page — صفحة التسعير (Desktop)

```
Design a premium pricing page for "تمعّن" Quranic contemplation app. Arabic RTL.

THEME: Dark (#15130f), gold accents (#f0e1c0), spiritual luxury feel. No flashy gradients — just refined simplicity.

CONTENT — 4 pricing tiers side by side:
1. عيدية (Gift): 28 SAR — "هدية تأمل ليوم واحد" — basic verse + reflection
2. المسار (Monthly): 82 SAR — "28 يوماً من التمعّن" — full daily experience, reflection journal, awareness tracking
3. التمكين (Annual): 820 SAR — "عام كامل من الحضور" — everything + book access + community + quarterly retreats
4. الخلوة (VIP): 8,200 SAR — "تحول شخصي عميق" — everything + 1-on-1 sessions + custom Gene Keys reading + private community

DESIGN DETAILS:
- Each tier: tall card with thin gold border, tier name in Amiri, price in large text, feature list with minimal dots
- The "التمكين" tier should be subtly highlighted (slightly brighter border, "الأكثر طلباً" label)
- Below cards: simple FAQ accordion
- Top: "اختر مسارك" heading + "كل مسار يبدأ بصمت" subtitle
- Guarantee badge: "ضمان ١٤ يوم — إن لم تجد ما تبحث عنه"
```

## 4. Progress Dashboard — لوحة التقدم (Desktop)

```
Design a personal progress dashboard for "تمعّن" meditation app. Arabic RTL.

THEME: Dark background, gold/warm accents. Data visualization should feel organic, not corporate.

LAYOUT:
- Top bar: day count "اليوم ١٤ من ٢٨", circular progress ring (gold fill)
- Journey track: horizontal timeline showing 28 days, completed days as lit gold dots, current day pulsing
- Phase indicator: current phase name (e.g. "هِبَة — الانفتاح") with color corresponding to progress
- 9 domains wheel: circular arrangement of 9 domain icons, each with fill level showing engagement
- Reflection history: list of past entries with date + first line preview, scrollable
- Awareness graph: simple line chart showing awareness scores over time, minimal axes
- Current streak: "١٤ يوماً متتالياً" with flame/light icon
- Bottom: next session CTA "ابدأ تأمل اليوم"

Keep it contemplative — this is a spiritual tracker, not a fitness app. Subtle animations, breathing transitions.
```

## 5. Mobile App — الشاشات الرئيسية (Mobile)

```
Design 4 mobile screens for "تمعّن" Quranic contemplation app. Arabic RTL. iPhone 15 Pro frame.

THEME: Dark (#15130f), gold (#f0e1c0), extreme minimalism. Bottom navigation with 4 items: الرئيسية، رحلتي، تأمل، روحي.

SCREENS:
1. HOME: Staff illustration (small), today's verse centered, "ابدأ تأمل اليوم" button, journey progress bar at bottom
2. MY JOURNEY: 28-day grid (4×7), completed days gold-filled, current day glowing, phase name at top
3. CONTEMPLATE: Full-screen verse with ambient glow, "اكشف الطبقة المخفية" button, swipe for reflection journal
4. SOUL: 9 domains in 3×3 grid, each with icon and name, tap to expand. Awareness meter at bottom.

Keep all text large and readable. Lots of breathing room. The app should feel like holding a sacred object.
```
